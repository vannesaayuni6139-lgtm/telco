## Cara berkontribusi (ringkas)

1. Fork repo ini ke akun GitHub Anda.
2. Clone fork Anda dan buat branch fitur / perbaikan:

```bash
git clone git@github.com:<username>/telco.git
git checkout -b feat/my-new-feature
```

3. Kerjakan perubahan, jalankan tes / build secara lokal.
4. Buat PR (pull request) ke `main` di repo upstream.
5. Tulis deskripsi PR yang jelas: tujuan, apa yang diubah, dan cara menguji.

## Gaya kode
- Ikuti pola TypeScript + React + Vite yang dipakai di repo.
- Gunakan Prettier / formatter yang Anda gunakan di project.

## Workflow PR
- Buka PR ke branch `main`.
- Satu PR sebaiknya memuat satu masalah/fitur jelas.
- Tulis unit test jika memungkinkan (atau langkah manual untuk memverifikasi perubahan).
